# Airbnb Assistant - Property Management Website

A modern Next.js website for Airbnb property management services, featuring virtual assistant solutions for property managers and hosts.

## Description

This project is a professional landing page for Airbnb Assistant services, helping property managers and hosts streamline their rental operations. The website showcases services, pricing plans, tools integration, client reviews, and more.

## Tech Stack

- **Framework:** Next.js 16
- **UI Library:** React 19
- **Styling:** Tailwind CSS 4
- **Language:** TypeScript

## Getting Started

### Prerequisites

- Node.js 18+ 
- npm or yarn

### Installation

```bash
# Install dependencies
npm install
```

### Development

```bash
# Run development server
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

### Build

```bash
# Create production build
npm run build

# Start production server
npm run start
```

### Linting

```bash
# Run ESLint
npm run lint
```
